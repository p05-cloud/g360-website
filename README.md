# G360 - Guru 360 Website

Official website for G360 (Guru 360) - Prakash More's fitness brand.

## About

### Key Features
- Multi-page responsive website
- Product catalog with filtering
- Brand showcase
- Team section (expandable)
- Contact form

## Features

### Navigation
- Responsive navbar with mobile hamburger menu
- Scroll effect on desktop
- Active link highlighting

### Product Filtering
- Filter by category: All, Protein, Amino Acids, Performance, Health & Wellness
- Smooth animations on filter change

### Animations
- Intersection Observer for scroll-triggered animations
- Smooth scroll for anchor links
- Hover effects on cards and buttons

### Forms
- Client-side validation
- Success/error messaging
- Pre-fill from URL parameters

## Deployment

This website is designed for static hosting on:
- Vercel
- Netlify
- GitHub Pages
- Any static file server

### Deploy to Vercel

1. Push to GitHub repository
2. Connect repository to Vercel
3. Deploy automatically

### Local Development

Simply open `index.html` in a browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
